# Bank Management System

## 📌 Description
A simple Bank Management System developed using C++. It allows users to perform basic banking operations like account creation, deposit, withdrawal, and balance checking.

## 🚀 Features
- Create account
- Deposit money
- Withdraw money
- Check balance

## 🛠️ Technologies Used
- C++
- File handling (if used)

## ▶️ How to Run
1. Clone the repository
2. Open in any C++ IDE (CodeBlocks / VS Code)
3. Compile and run `main.cpp`

## 📂 Project Structure
- main.cpp → Main program
- BankManagement.cbp → Project file
